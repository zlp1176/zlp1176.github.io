使用 hexo + github 搭建博客
发布于 2019-04-25 / 更新于 2019-04-25 / github / 热度 0 / 喜欢 0

　　一般发表文章我们会选择简书、掘金、博客园等平台，但是这些平台的样式过于单一，于是我选择搭建自己的博客网站。

一、基于版本

hexo　3.8.0
git　2.18.0

二、搭建过程

注：所有命令都在 git bash 运行，防止出错。(需要安装git、node)

2.1 关于github https://github.com

  1. 注册一个 github 账号(用户名将用于你的博客地址)
  2. 创建仓库 仓库名为 你的用户名.github.io
  3. 创建 ssh key, 配置到 github
    • 运行 cd ~/.ssh，检查本机已存在的 ssh 秘钥
    • 如果有秘钥，可以直接使用，否则需要创建新的秘钥
      https://help.github.com/en/articles/connecting-to-github-with-ssh
    • 运行 ssh -T git@github.com 测试是否成功

2.2 关于hexo http://hexo.io

  1. npm install -g hexo 来安装 hexo
  2. 新建空文件夹，进入文件夹，运行
    • hexo init (新建网站)
    • hexo g (生成静态文件)
    • hexo s (启动服务器)
  3. 这时 打开浏览器访问 http://localhost:4000 即可
  4. 上传到 github，需要修改 _config.yml
    deploy:
type: git
repo: git@github.com:你的github用户名/你的github用户名.github.io.git
branch: master
  5. npm install hexo-deployer-git --save
  6. hexo d 这时打开 https://你的用户名.github.io/ 就可以访问你的博客啦~
  7. 注意：
    • 如果过程中 hexo 命令报错，可以尝试将 node_modules 文件夹删除，然后运行 npm install 重新安装所需插件
    • 如果配置或博文修改不成功，可以 hexo clean 清理 public 内容，再来重新生成和发布
  8. 发布博文
    • hexo new “题目”
    • hexo d -g (生成并上传)

  9. 设置新的主题 https://hexo.io/themes/。(我选用的Annie)

    • 选择喜欢的主题后，git clone 地址 克隆到 theme目录下
    • 修改 _config.yml 中的theme: landscape 改为 theme: Annie
    • hexo g
    • 还要对相应主题进行配置，这个是我需要做的配置 https://sariay.github.io/2018/08/27/Annie%E4%B8%BB%E9%A2%98%E4%BD%BF%E7%94%A8%E8%AF%B4%E6%98%8E/
